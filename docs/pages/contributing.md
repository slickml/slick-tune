Please follow the full contributor workflow in the repository root:

**[CONTRIBUTING.md](https://github.com/slickml/slick-tune/blob/master/CONTRIBUTING.md)**

Quick checklist:

1. Open an issue before major work (`good-first-issue` is a great start).
2. Use `uv sync` and `poe format` / `poe check` / `poe test`.
3. Build docs with `uv sync --group docs && poe sphinx`.
4. Open a focused PR with the template checklist.
